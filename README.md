# Wu_vacuum_wannier_function
This repository contains the input and data files for the calculations done in https://arxiv.org/abs/2603.14105
To reproduce the data in the paper, please install JDFTx at jdftx.org and run the input files and implement the methods described in the paper.

The directories are explained below:
  data: contains the MTE and the rms spreads of the longitudinal energy distributions
  graphene: contains the JDFTx input files for the graphene calculations
  hBN: contains the JDFTx input files for the graphene calculations
  stability_tests: contains the wannier input files for the fcc and hcp VWF stability tests in vacuum
